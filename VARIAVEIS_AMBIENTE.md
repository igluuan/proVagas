# Variáveis de Ambiente

Esta aplicação utiliza variáveis de ambiente para configurar informações sensíveis.

## Banco de Dados

- `DB_PASSWORD`: Senha para o banco de dados PostgreSQL.
  - Exemplo: `DB_PASSWORD=sua-senha-segura`

## Administrador

- `ADMIN_PASSWORD`: Senha para a conta de administrador.
  - Exemplo: `ADMIN_PASSWORD=outra-senha-segura`

## JWT

- `JWT_SECRET_KEY`: Chave secreta para assinar os tokens JWT.
  - Exemplo: `JWT_SECRET_KEY=sua-chave-secreta-longa-e-segura`
